Coding Concepts I Used:

  • Concept 1: If/Else - This changes direction of my character and his movement all around, the key up and down codes interact with these as well.
  • Concept 2: Variables - The variables in my code control my character's appearance, as well as his location. Also helps with the background, and the soccer ball and goal.
  • Concept 3: The Red Blocks - All of the red blocks in my code are very important, such as set animation, velocityX, and rotation. All help with the soccer ball and my character.
  • Concept 4: Functions - The only function in my code is the draw function, which is the most important.

One Part of the Code I Understand Well:

The if/else and key down/up functions are really easy to understand. Because IF I press the down key then he will move a certain way, and if I hit the up key (which is the ELSE) he will go the other way.
